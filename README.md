# 💖 Declare Seu Amor — Porto Velho

> **"Com um simples gesto, você apoia projetos sociais em Porto Velho"**

Uma Landing Page moderna e de alto impacto para a campanha **Declare Seu Amor**, que incentiva os contribuintes a destinarem parte do seu Imposto de Renda (IR) para os Fundos Municipais da Criança, do Adolescente e da Pessoa Idosa de Porto Velho/RO.

---

## ✨ Funcionalidades

- 🚀 **Performance Extrema**: Desenvolvido com Next.js 16 e Turbopack para carregamento instantâneo.
- 📱 **Totalmente Responsivo**: Design fluido que se adapta perfeitamente a celulares, tablets e desktops.
- 🎨 **UI/UX Premium**: Estilo *Glassmorphism*, animações suaves com Framer Motion e tipografia moderna.
- 🔄 **Carrossel Dinâmico**: Galeria de imagens da campanha com navegação suave (Embla Carousel).
- 📋 **Passo a Passo Interativo**: Guia visual simplificado para orientar o contribuinte durante a declaração.
- ❓ **FAQ Inteligente**: Seção de perguntas frequentes com sanfona (accordion) para tirar todas as dúvidas.
- 🏠 **Impacto Local**: Seção dedicada a mostrar como o recurso fica no município e ajuda quem mais precisa.

## 🛠️ Tecnologias Utilizadas

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Estilização**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Ícones**: [Lucide React](https://lucide.dev/)
- **Carousel**: [Embla Carousel](https://www.embla-carousel.com/)
- **Fontes**: Tipografia customizada Pally (Local Font)

## 🚀 Como Executar o Projeto

### Pré-requisitos
- Node.js 20+ ou superior
- npm / yarn / pnpm

### Instalação
1. Clone o repositório:
```bash
git clone https://github.com/VgavaBR123/Site_Declare_Seu_Amor.git
```

2. Entre no diretório:
```bash
cd Site_Declare_Seu_Amor
```

3. Instale as dependências:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```
O site estará disponível em `http://localhost:3000`.
